# MITRE to Draw.io Exporter

Интерактивный инструмент для визуализации тактик и техник MITRE ATT&CK и экспорта выбранных элементов в формат схем Draw.io (mxGraph / XML). Также поддерживает работу с требованиями ФСТЭК России.

## Возможности

- **Матрица тактик и техник**: Полная русифицированная интерактивная таблица MITRE ATT&CK.
- **Поддержка субтехник**: Раскрывающиеся списки для детального выбора.
- **Экспорт в Draw.io**: 
  - Генерация готовых диаграмм для импорта в Draw.io (кнопка `MITRE draw.io`).
  - Поддержка экспорта в формате ФСТЭК (кнопка `FSTEC draw.io`).
- **Интеграция с PT NAD**: Подсветка "зеленых" техник (выявляемых PT Network Attack Discovery) на основе списков:
  - `pt_nad_green_techniques.txt`
  - `pt_nad_green_subtechniques.txt`
- **Проекты и импорт**:
  - Сохранение и загрузка конфигураций (проектов) в локальное хранилище браузера (LocalStorage).
  - Импорт существующих схем `.drawio`/`.xml` для восстановления состояния выбора.

## Использование

1. Откройте `index.html` в браузере.
2. Отметьте галочками интересующие техники и субтехники.
3. Нажмите кнопку **MITRE draw.io** или **FSTEC draw.io** для скачивания файла схемы.
4. Откройте [draw.io](https://draw.io/) и импортируйте скачанный файл.
